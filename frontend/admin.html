Настройки сайта

🖼️ Фоновые медиа для страниц

🎥 Фон для страницы «Видео»

Если загружается видео — укажите диапазон времени, который будет показываться на фоне. Для изображений время игнорируется.

📸 Фон для страницы «Галерея»

Для фотографий укажите позицию — где обрезать (CSS). 0% 0% = верх-лево, 50% 50% = центр, 100% 100% = низ-право.

🖼️ Фото для раздела «Биография»

Рекомендуемый размер: 480×520 px или больше (портретный формат).

🌄 Фон блока «Биография» (фоновое фото секции)

Фотография будет фоном блока «Биография» с тёмным оверлеем. Рекомендуется горизонтальный формат (min 1200×700 px).

🌄 Фон блока «Шоу» (фоновое фото секции)

Фотография будет фоном блока «Шоу» с тёмным оверлеем. Рекомендуется горизонтальный формат (min 1200×700 px).

🌄 Фон блока «Галерея» (фоновое фото секции)

Фотография будет фоном блока «Галерея» с тёмным оверлеем. Рекомендуется горизонтальный формат (min 1200×700 px).

🎪 Медиа для раздела «Шоу» (Главная)

Если медиа загружено, вместо иконки перетянет внимание на визуальную часть.

🖼️ Превью для видео Шоу (Постер)

Картинка, которая будет показываться до воспроизведения видео (иначе берется 1-й кадр).

📸 Фото для раздела «Галерея» (Главная)

🎫 Кнопка «Билеты» в меню

🗄️ Резервная копия базы данных

Скачивает текущую базу данных SQLite в виде файла juggler_backup.sqlite. Храните резервные копии в надёжном месте.
